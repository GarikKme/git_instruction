## git reset

**git reset** - Команда __git reset__, как можно догадаться из названия, используется в основном для отмены изменений. Она изменяет указатель **HEAD** и, опционально, состояние индекса. Также эта команда может изменить файлы в рабочей директории при использовании параметра --hard, что может привести к потере наработок при неправильном использовании, так что убедитесь в серьёзности своих намерений прежде чем использовать его.

### используйте команду:

 ```bash=
 git reset
 ```
